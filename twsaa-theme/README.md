# Twsaa Theme - قالب متجري لتوسع (Twsaa)

هذا المجلد يحتوي على نسخة مبدئية من قالب متوافق مع منصة توسع (Twsaa) مبني باستخدام Twig وPHP وCSS. القالب مُجهّز بالعربية بالكامل (RTL) وألوان مبدئية تناسب طلبك (أحمر غامق - أسود فاتح) ويمكنك تعديل أي عنصر بسهولة.

محتويات الحزمة:
- views/ : قوالب Twig (index.twig, product.twig)
- assets/css/style.css : أنماط القالب (قابلة للتعديل عبر متغيرات CSS)
- assets/images/ : مكان لحفظ صور المنتجات والشعار
- data/products.json : مثال لمنتجات أقل من 20 منتجًا (يمكن استبداله أو استيراده لوحة توسع)
- index.php : نقطة الدخول لعرض القالب محليًا (مستخدم فقط للمعاينة)
- composer.json : تعريف تبعيات Twig (إن أردت المعاينة محليًا)

طريقة النشر على منصة توسع (موجز):
1. ضُمّ مجلد الحزمة (الموجود في هذا الريبو تحت `twsaa-theme/`) إلى ملف ZIP.
2. سجّل دخولك إلى لوحة توسع > الثيمات (Themes) > رفع قالب جديد (Upload Theme) واختر ZIP.
3. بعد الرفع، عيّن القالب كافتراضي ثم افتح إعدادات القالب لتعيين المنتجات أو استيرادها.

ملاحظات سريعة:
- العملة مُثبّتة في القالب كـ SAR، ويمكن تغييرها في `views/_partials/currency.twig` أو عبر إعدادات توسع.
- صور المنتجات: ضع صورك في `assets/images/` بنفس اسم الملف المشار إليه في `data/products.json`، أو استبدل مسارات الصور في `data/products.json` بمسارات حقيقية.
- بوابة الدفع: كما ذكرت، سيتم إعداد بوابة الدفع من داخل لوحة توسع — لا حاجة لتعديل القالب إلا إن تطلب مزوّد الدفع إجراء تكامل خاص.

إذا رغبت أرفع ملف ZIP جاهز بالثيم إلى هذا الريبو كبنشر (release) وأساعدك بخطوات رفعه داخل لوحة توسع مباشرة.
